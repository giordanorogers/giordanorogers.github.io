Neural Networks

Neural networks are an algorithm for transforming data tensors to discover patterns and thereby predict a given output for given input. They act as universal function approximators. Neural networks consist of a series of weighted tensor multiplications with some nonlinear activation functions. They learn to filter out the noise and infer the essential patterns in the data. In this way they are compressing what is necessary about the data to predict things about it. This can be predicting the classification of the data, predicting what data would come next, transforming data from one type to another, and many other things. PyTorch is s framework in Python written on numpy and C which allows us to use GPUs to train neural network models in parallel and with simplified abstract classes and methods.

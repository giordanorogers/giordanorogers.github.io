# Axioms of my Epistemology

1. Axioms cannot be true or false, only helpful, neutral, or harmful.
2. Any statement which is consistent with the axioms of a system is true within that system.
3. Axiomatic consistency allows for inductive reasoning.
4. Inductive reasoning is the process of reaching novel conclusions from established premises.
5. Inductive reasoning allows us to expand our understanding.

---

1. Truth does not necessitate agreement or acknowledgement.
2. All tautologies are true.
3. This implies that truth results, at least in one form, from axiomatic consistency.
4. To define truth as axiomatic consistency implies both that you cannot have truth that isn't building on an 
   established set of axioms. Can a truth negate or evaluate its own axioms?
5. I'm dissatisfied with this definition because, for example, if you use a probabilistic set of axioms, as in language 
   embeddings, where the fundamental relationship between concepts are derived rather than instantiated, then this 
   definition either breaks the notion that anything in embedding space can be considered true, as it has no axioms 
   with which to be consistent unless axioms are assigned afterward.
